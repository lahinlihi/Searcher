## 🔍 대시보드 표시 관련

### 현재 상황
IRIS 공고는 데이터베이스에 정상 저장되었으나, 대시보드 첫 페이지에는 표시되지 않습니다.

### 원인
**스마트 정렬 알고리즘**:
1. 마감일이 지나지 않은 공고 우선
2. **관심 키워드 매칭 점수 높은 순** ← 여기서 밀림
3. 공고일 최신순

현재 설정된 관심 키워드:
```
['AI', '인공지능', '시스템', '디지털', '개발', 'IT', 'AX', '전환']
```

IRIS 공고 제목에 위 키워드가 없어서 매칭 점수가 0점이 되어 정렬 순서에서 뒤로 밀렸습니다.

- `연구`
- `과제`
- `지원사업`

- 통계 섹션에서 "IRIS: 10건" 확인 가능

### 방법 3: 직접 검색
- 검색 페이지에서 키워드 입력하여 필터링
- 예: "연구", "과제", "지원" 등
