## 3. 왜 DB가 비어있는가?

### 가능한 원인

#### 1. **자동 크롤링이 실제로 실행되지 않음**
- 스케줄러는 매일 09:00, 17:00에 실행 설정
- 오늘 17:23에 크롤링 로그가 있지만 0건 수집
- **결론**: 실행은 되지만 데이터를 못 가져옴

#### 2. **모든 사이트에서 0건 수집**
- 각 사이트 크롤링이 실패하거나
- 셀렉터가 잘못되어 데이터를 못 가져옴
- API 응답 오류

#### 3. **크롤링은 성공했지만 DB 저장 실패**
- 가능성 낮음 (코드 확인 결과 저장 로직 정상)
