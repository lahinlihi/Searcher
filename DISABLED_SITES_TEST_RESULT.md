## 🔧 공통 문제점

### 1. link 셀렉터 누락 (6/6)
**모든 사이트**에서 link 셀렉터가 설정되지 않음

**영향**:
- 크롤러가 링크를 추출할 수 없음
- 공고 상세 URL이 없어 불완전한 데이터

**해결 방법**:
```json
"selectors": {
  "item": "table tbody tr",
  "title": "td a",
  "link": "td a"  // 추가 필요
}
```

### 2. 셀렉터 불일치 (4/6)
Selenium을 사용하는 사이트들(fanfandaero, motie, msit, nia)에서 셀렉터가 맞지 않음

**가능한 원인**:
1. 사이트 구조가 변경됨
2. 셀렉터가 처음부터 잘못 설정됨
3. JavaScript로 동적 로드되는 콘텐츠 타이밍 문제

**해결 방법**:
1. 웹 브라우저에서 F12 개발자 도구로 실제 HTML 구조 확인
2. 올바른 셀렉터 찾기
3. `data/settings.json`에서 셀렉터 수정

### 3. 페이지 접근 실패 (2/6)
SSL 검증을 비활성화한 사이트들(koica, kosmes)에서도 접근 실패

**가능한 원인**:
1. 사이트가 다운되었거나 URL 변경
2. 네트워크 방화벽 또는 차단
3. User-Agent 차단

**해결 방법**:
1. 웹 브라우저에서 직접 URL 접속 확인
2. URL이 변경되었으면 `data/settings.json`에서 수정
3. 사이트가 차단하는 경우 Selenium 사용 고려
