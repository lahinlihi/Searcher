## 🔒 비활성화된 사이트 (6개)

- **이유**: SSL 프로토콜 오류 (SSL: UNEXPECTED_EOF_WHILE_READING)
- **기술적 문제**: 서버 측 SSL/TLS 설정 문제
- **해결 방법**: 없음 (서버 측 수정 필요)

### 2. 한국지능정보사회진흥원 (nia)

- **기술적 문제**: 별도 API 서버에서 JSON 데이터 로딩
- **해결 방법**: API 역분석 필요 (2-4시간 소요)

### 3. 과기정통부 (msit)
- **이유**: AJAX 동적 데이터 로딩, 실제 게시판 데이터 미존재
- **기술적 문제**: JavaScript로 API 호출 후 렌더링
- **해결 방법**: API 역분석 필요 (2-4시간 소요)

### 4. 한국중소벤처기업유통원 (fanfandaero)

- **해결 방법**: API 역분석 필요 (2-4시간 소요)

### 5. 산업통상부 (motie)

## 🔧 구현된 기술

### 1. Selenium 통합 ✅
- **목적**: JavaScript 동적 페이지 크롤링
- **기능**:
  - Chrome WebDriver 자동 설치
  - 헤드리스 모드 (백그라운드 실행)
  - SSL 인증서 오류 우회
  - 페이지 로딩 대기 및 스크롤 시뮬레이션
- **적용 사이트**: seoul-city, koica

### 2. SSL 검증 우회 ✅
- **목적**: SSL 인증서 문제 사이트 지원
- **적용**: verify_ssl=False 옵션

### 3. 범용 크롤러 (Generic Crawler) ✅
- **목적**: 설정 기반 다양한 사이트 크롤링
- **기능**:
  - CSS 셀렉터 기반 데이터 추출
  - 자동 URL 정규화
  - 날짜 파싱
  - 에러 핸들링

### 4. API 크롤러 ✅
- **목적**: 공공데이터포털 API 직접 호출
- **기능**:
  - REST API 호출
  - 페이지네이션 자동 처리
  - XML/JSON 파싱
- **적용**: 나라장터 API
